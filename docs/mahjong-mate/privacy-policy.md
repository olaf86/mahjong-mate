# プライバシーポリシー（雀メイト / Mahjong Mate）

最終更新日: 2026-02-22

本ポリシーは、アプリ「雀メイト」（以下「本アプリ」）における利用者情報の取り扱いを定めるものです。

## 1. 取得する情報

本アプリは、以下の情報を取り扱います。

### 1-1. Firebase Authentication に関する情報

- ユーザーID（UID）
- メールアドレス（メールアドレス登録・ログイン時のみ）
- 認証状態（匿名ユーザー / メールログインユーザー）

### 1-2. Cloud Firestore に保存される情報

- ルールセット情報
  - ルールセット名、説明、ルール内容、公開設定、更新日時
  - オーナー名
  - オーナーUID
  - 共有コード（公開時）
- フォロー情報
  - フォローしたルールセットID
  - 並び順
  - フォロー日時

### 1-3. 端末内（ローカル）に保存される情報

- 表示名（オーナー名設定）
- 「共有リンクを開いたら自動でフォローする」設定

## 2. 利用目的

取得した情報は、以下の目的で利用します。

- 本アプリの認証機能の提供
- ルールセットの作成・保存・表示・共有
- フォロー機能および表示順管理
- 設定内容の保存
- 不正利用の防止およびサービス品質の維持

## 3. 第三者サービス

本アプリは、以下の第三者サービスを利用します。

- Google Firebase Authentication
- Google Cloud Firestore

これらのサービスにおけるデータ取扱いは、Google のプライバシーポリシー等に従います。

## 4. 情報の第三者提供

開発者は、法令に基づく場合を除き、利用者情報を第三者へ提供しません。  
ただし、利用者が公開設定で作成したルールセットは、共有コード・共有リンクを通じて他の利用者が閲覧できます。

## 5. データの保持と削除

- 利用者が作成したデータは、利用者操作（削除等）または開発者の運用上必要な対応があるまで保持されます。
- アプリのログアウト後は匿名ユーザーとして再ログインされる場合があり、以前のアカウントデータは再ログインするまで表示されません。
- 利用者データの削除を希望する場合は、下記連絡先へお問い合わせください。

## 6. セキュリティ

開発者は、アクセス制御など合理的な安全管理措置を講じ、情報の漏えい・滅失・毀損の防止に努めます。

## 7. 未成年の利用

未成年の方は、必要に応じて保護者の同意を得たうえで本アプリを利用してください。

## 8. ポリシーの変更

本ポリシーの内容は、必要に応じて変更することがあります。  
重要な変更を行う場合は、アプリ内または公開ページ上で告知します。
