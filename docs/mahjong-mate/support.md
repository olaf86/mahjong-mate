# サポート（雀メイト / Mahjong Mate）

最終更新日: 2026-03-02

本ページは、アプリ「雀メイト」のサポート情報をまとめたものです。

## お問い合わせ

不具合報告・改善要望は、以下からご連絡ください。

- [GitHub Issues](https://github.com/olaf86/mahjong-mate/issues)

報告時には、次の情報を含めてください。

- 端末OS（iOS / Android）とバージョン
- アプリのバージョン
- 発生手順
- 実際に起きた内容
- 期待していた動作

## よくある質問（FAQ）

### Q. ルールセットが一覧に表示されません。

- ログイン状態（匿名 / メール）を確認してください。
- 共有コードで開いた場合は、フォロー操作が必要な場合があります。

### Q. オーナー名を変更したのに表示が変わりません。

- 設定画面でオーナー名を保存後、画面を再表示してください。
- 通信状態によって反映に時間がかかる場合があります。

### Q. 機種変更時にデータを引き継げますか？

- メールアカウントでログインしていれば、同じアカウントで再ログインすることで継続利用できます。
- 匿名ログインのみの場合、端末変更やアプリ削除でデータが失われる可能性があります。
